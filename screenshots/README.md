# Screenshots

Add your project screenshots here:

- `login.png` - Login page screenshot
- `dashboard.png` - Dashboard/main interface screenshot  
- `about.png` - About page screenshot

## How to add screenshots to GitHub:

1. Take screenshots of your application
2. Save them as PNG files with the names above
3. Upload them to this folder via GitHub web interface or:
   ```bash
   git add screenshots/login.png
   git add screenshots/dashboard.png
   git add screenshots/about.png
   git commit -m "Add project screenshots"
   git push
   ```
